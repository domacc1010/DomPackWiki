# Where to Get Information

## Pokémon Section
Combine several authoritative sources rather than relying on one:
- **Official Cobblemon Wiki** — mechanics, Pokémon data, items, spawning
- **Cobblemon's GitLab/GitHub repos** — JSON data for Pokémon, moves, spawn
  rules, drops, evolutions (best source for bulk/automated extraction)
- **Pokémon Showdown** — competitive movesets and battle data
- **Bulbapedia** — canonical Pokémon information
- **Serebii** — locations, abilities, learnsets, Pokédex details

## Non-Pokémon Mods
- **CurseForge project pages** — features and changelogs
- **Modrinth project pages** — documentation and version compatibility
- **GitHub repositories** — configuration options, commands, advanced usage
- **Mod authors' own wikis/docs** — when available

## Automation Note
Long-term, most of the Pokémon section (types, abilities, evolutions, moves,
spawn conditions, drops) can be generated programmatically by extracting data
straight from the Cobblemon JSON files and your own server datapacks. That
leaves the writing effort focused on what a script can't produce: server-
specific strategy, lore, and progression content.
