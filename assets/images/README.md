# Images

Drop screenshots, renders, and other images here (subfolders are fine, e.g.
`assets/images/pokemon/`, `assets/images/builds/`). Reference them from any
page with the `.img-frame` wrapper — see the root README for the snippet.

Keep filenames lowercase-with-dashes and reasonably sized (compress large
screenshots before adding — a static site with unoptimized 8MB PNGs will
load slowly for players on mobile data).
